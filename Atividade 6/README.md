Com o padrão Builder podemos simplificar o código, o que torna a codificação mais simples, para o desenvolvedor e
outro profissional que poderá utilizar o código.

Um ponto positivo é poder separar a construção de um objeto de sua representação de modo que o mesmo processo de construção possa criar diferentes representações.

Essa metodo é parecido com o Abstract e Factory, mais o builder permite separar a construçoes de um objeto em métodos menores. Utiliza objetos específicos para tratar problemas comuns,fica mais fácil manter
o foco da responsabilidade das classes.
